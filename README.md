# Rust Essentials Learning Plan

This project provides a comprehensive guide to Rust programming from basic to advanced concepts, designed for learners starting from zero.

## Repository Structure

- **`src/`** — Runnable code examples organized by topic
- **`exercises/`** — Practice exercises for each topic (with solutions)
- **`explanations/`** — Detailed written explanations of each concept
- **`references/`** — Links to official Rust documentation and resources
- **`README.md`** — This file

## How to Use

1. Read the concept explanation in `explanations/`
2. Run the corresponding examples in `src/`
3. Complete the exercises in `exercises/` before checking solutions
4. Refer to `references/` for official documentation

```bash
# Run an example
cargo run --bin variables_and_mutability

# Run tests
cargo test
```
